Client Projector

Open This Page and Show on Projector

❌ #1 Setup project following these instructions. Make sure node server is running on the Server.

❌ #2 and 3 Run the "Tracking Server" on the same network or device as this "Client Projector." (Test to see if right hand control is visible on network.)

❌ #4 Show this "Client Projector" page in full screen but NOT in a WebVR browser. Make sure no toolbar, bookmark or other browser bars are visible.

❌ #5 Use the right HTC Vive controller to locate the red target marker shown in your projector. Align the controller marker to match and press the `trigger` button.

❌ #6 Press the "Init" button below.

After you've setup the `server`, project this `client` page down on the floor or surface. Click `init` once `server` has matched the red target marker, then use controls below to calibrate.
