Tracking Server

Open This Page in a WebVR Browser with HTC Vive

❌ #1 Setup project following these instructions. Run node server on this computer.

❌ #2 Run this "Tracking Server" page using a WebVR browser such as supermedium with HTC Vive and controllers.

❌ #3 Now press the "Enter VR" icon in the lower right corner to enter VR. (This gives the app access to headset and controller positions.)

#3.25 Make sure you have 2 Vive controllers and 1 Tracker. Make sure SteamVR has assigned "handedness."

❌ #3.5 Make sure the left controller is visible to scene.

❌ #4 In a separate browser (or even computer) load the "Client" page and display it on your projector in full screen.

❌ #5 Use the left HTC Vive controller to place the virtual projector in the identical location as the real-life projector and press the `trigger` button.

❌ #6 Then, on the "Client" follow the setup instructions on the screen and click the green "Init" button.
